# 🌍 Sandy Geo Journey

An interactive geospatial CV that maps my professional journey through space, data, and storytelling.

## 🧠 About the project

This project is a web-based StoryMap built using geospatial technologies. It combines mapping, personal narrative, and interactive design to present my professional profile as a Geomatics Engineer.

Instead of a traditional CV, this platform allows users to explore my experience spatially and visually.

## 🚀 Technologies used

- OpenLayers (web mapping)
- GeoServer (WFS services)
- HTML, CSS, JavaScript
- PostgreSQL / PostGIS

## 🗺️ Features

- Interactive map with geospatial data
- Custom sidebar navigation
- Dynamic zoom to locations
- Popups with professional information
- Visual storytelling with images

## 🎯 Purpose

To demonstrate my skills in:

- Geospatial data management
- Web GIS development
- Spatial thinking and storytelling
- Frontend + backend integration
